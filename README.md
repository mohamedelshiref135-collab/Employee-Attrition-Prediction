# Employee Attrition Prediction

This project predicts whether an employee will leave the company using Machine Learning.

## 🚀 Project Overview
The model analyzes employee data such as:
- satisfaction
- number of projects
- salary level
- department
- working hours
- bouns
- reviw
- tenure

Then predicts if the employee is likely to leave.

## 🧠 Model
The machine learning model was trained using Python and scikit-learn.

## 🖥️ Web Application
The model is deployed using Streamlit to provide an interactive web interface.

## 📂 Project Files
- Attrion_web.py → Streamlit web app
- My_Project_Employee_Attrion.ipynb → Trained ML model
